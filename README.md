# 🔴 AMD GPU Computing & ROCm Exploration

> Exploring GPU-accelerated computing using AMD ROCm platform — from fundamentals to machine learning workloads.

---

## 👋 About This Repository

This repository documents my learning journey into **GPU computing** using AMD's open-source software stack, **ROCm (Radeon Open Compute)**. As a software developer and student, I'm passionate about understanding how modern GPUs can accelerate computational workloads — from scientific computing to deep learning.

---

## 🎯 Goals

- Understand the fundamentals of GPU architecture and parallel computing
- Learn AMD ROCm stack and HIP (Heterogeneous-Compute Interface for Portability)
- Run deep learning experiments using PyTorch with ROCm backend
- Explore GPU-accelerated data processing and scientific computing
- Contribute to open-source projects in the GPU/ML ecosystem

## 📂 Planned Projects

### 1. ROCm Environment Setup
> Step-by-step guide to setting up AMD ROCm on various Linux distributions and verifying GPU functionality.

### 2. HIP Fundamentals
> Learning HIP (AMD's CUDA-compatible API) with basic parallel computing examples — vector addition, matrix multiplication, and more.

### 3. PyTorch + ROCm Deep Learning
> Running and benchmarking PyTorch models on AMD GPUs, including training simple neural networks and fine-tuning open-source models.

### 4. GPU Performance Benchmarking
> Comparing CPU vs GPU performance on compute-intensive tasks using AMD hardware through the AMD Developer Cloud.

### 5. Open-Source Model Experiments
> Running inference and training experiments on publicly available models (e.g., from HuggingFace) using AMD GPU infrastructure.
